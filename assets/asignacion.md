
# Práctica Lorem Ipsum
El alumno deberá construir un sitio con HTML + CSS + JS similar a Lorem Ipsum, toma como referencia la [imagen](./Lorem%20Ipsum.png), puedes toma como referencia tu archivo creado en la Unidad_1 o visita el sitio de [Lorem Ipsum](https://www.lipsum.com/)

---

##  Requerimientos HTML

En el archivo `lorem.html` se debe cumplir con lo siguiente:

1. Declarar correctamente el **DOCTYPE HTML5**
2. Usar etiquetas semánticas:
   - `<header>`
   - `<main>`
   - `<section>`
   - `<footer>`
3. Incluir:
   - Un encabezado principal con el título **Lorem Ipsum**
   - Un subtítulo o cita debajo del título
   - Dos columnas informativas:
     - **What is Lorem Ipsum?**
     - **Why do we use it?**
   - Una sección de generación de texto que incluya:
     - Input numérico
     - Radio buttons (paragraphs / words)
     - Botón de generar texto
     - Área de texto para mostrar el resultado
4. Vincular correctamente los archivos CSS y JS

---

##  Requerimientos CSS

En el archivo `styles.css` el alumno debe:

- Aplicar una **tipografía serif** para el título principal
- Centrar el encabezado principal
- Usar **layout de dos columnas** (Flexbox o Grid)
- Mantener:
  - Espaciado limpio
  - Buena legibilidad
  - Colores neutros
- Estilizar:
  - Botón
  - Inputs
  - Área de texto
- Asegurar que el diseño sea **responsive**

---

##  Requerimientos JavaScript

En el archivo `script.js` se debe:

1. Capturar eventos del botón **Generate Lorem Ipsum**
2. Leer:
   - Cantidad ingresada
   - Tipo seleccionado (párrafos o palabras)
3. Generar texto Lorem Ipsum dinámicamente
4. Mostrar el texto generado dentro del `<textarea>`
5. Validar:
   - Que el número sea mayor a 0
   - Que exista una opción seleccionada

---

##  Funcionalidades Mínimas Esperadas

-  El botón genera texto
-  El texto aparece en pantalla
-  Se puede elegir entre párrafos o palabras
-  La interfaz es clara y ordenada
-  El código está correctamente indentado

---

##  Buenas Prácticas

- Usar nombres de variables descriptivos
- Separar responsabilidades (HTML / CSS / JS)
- Evitar código duplicado
- Comentar el código JavaScript cuando sea necesario
